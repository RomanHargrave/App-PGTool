NAME
====

App::PGTool - Work with ProGuard Map Data

SYNOPSIS
========

Run pgtool

DESCRIPTION
===========

App::PGTool is a suite of code and tools for working with ProGuard map data. Often it might prove helpful to be able to edit and transform these maps.
